Configuration Studies for a 5-Tonne Class LOX–Methane Upper Stage Engine

Overview

This project focuses on configuration studies for a 5-tonne class LOX–LCH4 (Liquid Oxygen–Liquid Methane) upper-stage rocket engine. The study investigates suitable engine cycles and evaluates performance characteristics to support efficient upper-stage propulsion system design.

Objectives

- To identify the most suitable engine cycle for a LOX–LCH4 upper-stage engine with a nominal thrust of 50 kN.
- To analyze the capability of throttling between 50% and 110%, constrained by a maximum nozzle exit diameter of 1 m.

Tools and Software Used

- AutoCAD
- NASA CEA (Chemical Equilibrium with Applications)
- NIST REFPROP
- Microsoft Excel
- Microsoft Word
- Microsoft word 
- Easy Plot

Methodology

- Selection and comparison of candidate engine cycles.
- Propellant performance analysis using NASA CEA.
- Thermodynamic property evaluation using NIST REFPROP.
- Data analysis and visualization using Excel and Easy Plot.

Key Outcomes

- Comparative assessment of suitable LOX–LCH4 engine cycles.
- Evaluation of throttling performance from 50% to 110% thrust levels.
- Identification of a feasible upper-stage engine configuration as Electric Expander cycle within design constraints.
